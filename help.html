This page provides some basic hints on how to use this service.

Input the server address into the box (usually 127.0.0.1). You should see that a 'connection successful' message is displayed.

If the connection has failed, please ensure that the server is active and operational and that the address hasn't changed.

Once the server is connected, simply drag and drop the files onto the interface.
Files are automatically sent to the server for storage and processing.

To exit the application, click the menu and select 'Exit'.
